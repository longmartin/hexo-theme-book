hexo-theme-book
=================

Material Design 风格的Hexo主题，基于 **Hexo 3.0+** 制作。

[Preview](http://blog.intenbelief.top)

## Feature

1. 全端响应式主题。
2. 添加了英文字体支持 bluehost。
3. 无后端依赖的分享、浏览统计功能实现。
4. 完美集成valine以及gitment评论系统。
5. 基于静态数据的站内搜索，无第三方侵入。
6. 支持文章打赏。
7. 文章内可引入Github仓库。
